Массив (Array)
-
Массив — это структура данных, представляющая собой упорядоченную коллекцию элементов одного типа, хранящихся в непрерывной области памяти, доступ к которым осуществляется по индексу.

Основные характеристики массива:
-
Фиксированный размер (в большинстве языков)

Непрерывное расположение в памяти

Прямой доступ к элементам по индексу за время O(1)

Однородность типов — все элементы одного типа

Стек (Stack)
-
Стек — это структура данных, работающая по принципу LIFO (Last In, First Out — «последним пришёл, первым ушёл»), где элементы добавляются и удаляются только с одного конца (вершины стека).

Основные характеристики стека:
-
LIFO — последний добавленный элемент удаляется первым

Одна точка доступа — вершина стека (top)

Динамический размер — может расти и уменьшаться

Ограниченный доступ — нельзя получить произвольный элемент
