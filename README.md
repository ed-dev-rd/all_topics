Функция generate_binary_strings
Это рекурсивная функция, которая генерирует все возможные бинарные строки (состоящие из символов '0' и '1') заданной длины n.

Параметры:

n: Целое число, определяющее длину генерируемых бинарных строк
current: Текущая формируемая строка (используется в рекурсии)
result: Список для накопления результатов

def generate_binary_strings(n, current="", result=None):
    if result is None:
        result = []
    
    # Базовый случай рекурсии
    if len(current) == n:
        result.append(current)
        return result
    
    # Рекурсивный случай: добавляем '0' и '1'
    generate_binary_strings(n, current + "0", result)
    generate_binary_strings(n, current + "1", result)
    
    return result
  
Описание алгоритма:
-

Базовый случай рекурсии:

if len(current) == n:
    result.append(current)
    return result
    
Когда длина текущей строки достигает целевой длины n, строка добавляется в результат и рекурсия завершается.

Рекурсивный случай:

generate_binary_strings(n, current + "0", result)

generate_binary_strings(n, current + "1", result)

Функция вызывает саму себя дважды: один раз добавляя '0' к текущей строке, и второй раз добавляя '1'. Это создает бинарное дерево рекурсии.

Временная сложность и ее анализ
-

Временная сложность: O(2^n × n)

Что влияет на временную сложность:

Количество строк: 2^n

Для каждой позиции в строке длины n есть 2 варианта выбора ('0' или '1')
Общее количество бинарных строк: 2 × 2 × ... × 2 (n раз) = 2^n
